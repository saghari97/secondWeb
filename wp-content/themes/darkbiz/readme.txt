== Description ==
DarkBiz, a multipurpose WordPress theme, that has a dark bold look which will give a unique modern website. This theme is SEO friendly , cross browser compatiable, Gutenberg Support and fully translation ready. It includes, Custom Header, Featured Slider, Services section, About,  Team, Counters, Blog, Footer widgets etc. This theme can also be fit for creative, corporate, business,  blogging, consulting, medical, ecommerce, fashion, Online News,  yoga, charity, , travel, education and any other multiple business needs. is fully compatible with Gutenberg WordPress Blocks. With Rise Blocks and DarkBiz, you are able to create unique and beautiful layouts for your blog or websites with zero knowledge in coding.  You can easily import and export the json block files to build inner pages. Our dedicated team will help you in setting up websites. Follow the links for instructions. Full Demo: https://preview.risethemes.com/sites/darkbiz/

For more information about Darkbiz please go to https://risethemes.com/

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
